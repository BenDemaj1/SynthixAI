SynthixAI Support

App Description

SynthixAI is an AI-based image editing app for automatic background removal.

System Requirements

Device Compatibility

  • Supports: iOS devices with ARM64 architecture
  • Compatible with iPhones from iPhone 5s (2013)
  • Compatible with iPads from iPad Air 2 (2014)

Minimum iOS Version

  • Required: iOS 12.0 or higher
  • Recommended: Latest iOS version

Storage Requirements

  • App Size: Approximately 28 MB

Supported Devices

  • iPhone
  • iPad
  • iPod Touch with compatible iOS

Usage

  • Photos uploaded by clicking on the image container in the app
  • Supported image formats:
    • JPEG (.jpg, .jpeg)
    • PNG (.png)
    • WebP (.webp)
    • TIFF (.tiff)
  • AI background removal processed server-side
  • Processing through local AI model on the server

Contact

Email: arbendemaj@hotmail.de

Support requests answered within 48 hours

Privacy

  • Uploaded images automatically deleted after processing
  • Image processing occurs exclusively in working memory
  • No permanent storage of user images
  • Processing is confidential and temporary
  • No user data shared with third parties
